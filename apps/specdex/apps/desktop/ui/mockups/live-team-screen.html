Live-team screen + DRAMS scrollbars

Two related upgrades. Open in light and dark, scroll every pane (vertical and horizontal), and pick a direction. Every terminal below is independently scrollable so the scrollbar treatment is actually visible.

Scrollbars
One reusable "soft hardware" bar — thin, no track, a quiet --rule pill that darkens on hover. Replaces the chunky native OS bars on panes, the project sidebar, and tall lists. See the native-vs-DRAMS comparison first.
L-A
Two columns — coder | reviewer side by side, each a full-height scrollable terminal with an agent header strip and a Model/Ctx status footer.
L-B
Stacked tall — panes stacked vertically, each generously tall, the page scrolling as a whole.
L-C
Focused + switcher — one big terminal with a segmented dex-coder / dex-reviewer switch to flip which agent fills the screen.
Pick
L-A (two-column) for the standard 2-agent team — both agents visible at once, no flipping. Fall back to L-C only if a team ever exceeds two agents.
