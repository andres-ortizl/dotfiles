Topbar + legend — layout options for the fleet list view

Each panel below is an independently resizable mini-window. Drag the right edge of any frame inward to ~480px and watch only that frame re-layout (panels react to their own width via CSS container queries, not the browser window). Default width is deliberately narrow (~620px) so the responsive behavior is visible immediately — the whole point is "not full screen."
